My game for JS13kgames

I built a simple game for js13kgames using html,css and js, Jun 18, 2019

Js13kgames...

The game consists of the following rules.
1. The size of the game should be less than or equal to 13kb.
2. The game should be developed mainly using html,js and css.
3. There is a new theme every year.

The game is a simple type-writing game. There will be a canvas. Random letters will come down. If we type the letter the corresponding letter will vanish on the canvas.A new letter will emerge on the line of the old letter. Letters are generated from A-Z randomly using a random function. The letters will start coming down with an initial speed. As the game progresses the speed of the letters that comes down also increases. Once the letter hits the bottom of the screen i.e, the canvas, the Game is over!!
